# 디자인 스타일 프롬프트 — "Dark Brutalist Editorial / Citron Accent"

아래 프롬프트를 그대로 복사해서 UI 제작 요청에 붙여 쓰면 동일한 톤앤매너의 결과물을 얻을 수 있습니다.
콘텐츠(섹션 구성, 카피)만 채워서 사용하세요.

---

## 🟢 복사용 프롬프트 (전체)

다음 디자인 시스템을 정확히 따르는 [페이지/컴포넌트 종류]를 만들어줘.

**전체 무드**
다크 브루탈리스트 에디토리얼 스타일. 잡지 레이아웃처럼 거대한 타이포그래피, 넓은 여백, 얇은 헤어라인 구분선으로 구성. 따뜻한 검정 배경 위에 크림색 텍스트, 그리고 단 하나의 형광 시트론 컬러를 액센트로 사용. 테크니컬하면서도 자신감 있고 미니멀한 분위기.

**컬러 팔레트 (CSS 변수로 정의)**
- `--ink: #0b0b09` — 메인 배경 (따뜻한 검정)
- `--ink-2: #121210` — 보조 배경 (약간 밝은 검정, 띠/블록용)
- `--paper: #f4f1e6` — 메인 텍스트 (크림 화이트)
- `--paper-dim: #a8a596` — 보조 텍스트 (뮤트된 그레이지)
- `--citron: #ccff00` — 액센트 (형광 시트론/샤르트뢰즈) — 강조·호버·하이라이트에만 절제해서 사용
- `--citron-deep: #9fc800` — 시트론 딥 버전
- `--line: rgba(244,241,230,0.14)` — 헤어라인 구분선 (얇고 은은하게)

**타이포그래피**
- 본문 폰트: Pretendard (한글 최적화), 기본 weight 500(Medium), line-height 1.6
- 모노 폰트: JetBrains Mono — 라벨, 메타데이터, 내비게이션, 숫자, 기술적 텍스트에 사용. 대문자 + letter-spacing 0.1em 내외
- 디스플레이 헤딩: weight 800, `clamp(56px, 15vw, 200px)` 같은 초대형 사이즈, letter-spacing -0.04em(타이트), line-height 0.86(아주 좁게)
- 일부 단어는 외곽선 처리: `-webkit-text-stroke: 2px var(--paper); color: transparent;`
- 강조 단어는 시트론 컬러로
- 섹션 라벨은 모노 + 대문자 + 번호 프리픽스 형식 (예: `001 — [라벨]`, `002 — [라벨]`), 앞에 짧은 시트론 라인(`::before` 30px)

**레이아웃**
- 컨테이너: max-width 1200px, 좌우 패딩 32px, 가운데 정렬
- 섹션 상하 패딩 넉넉하게 (130px 내외)
- 비대칭 그리드 활용 (예: 1.4fr 1fr)
- 콘텐츠 구분은 박스 대신 1px 헤어라인(`--line`) 위주
- 통계/지표는 테이블처럼 테두리로 칸 나눈 그리드
- 리스트는 번호 매긴 인덱스 형식 (`/01`, `/02` … + 타이틀 + 우측 설명)

**시그니처 디테일 (가능하면 포함)**
1. **필름 그레인 오버레이**: 전체 화면에 SVG fractalNoise 노이즈를 `opacity:.05; mix-blend-mode:overlay`로 고정 배치
2. **커스텀 커서**: 14px 시트론 원형 커서가 마우스를 부드럽게(lerp 0.2) 따라오고, 링크/인터랙션 요소 호버 시 64px로 커짐. `mix-blend-mode:difference` 적용. 터치 기기에선 비활성화
3. **마퀴 띠**: 키워드가 무한 가로 스크롤되는 밴드. 단어를 채움/외곽선(stroke) 번갈아 표시하고 사이에 시트론 별표(✳) 구분자
4. **스크롤 리빌**: 요소가 뷰포트 진입 시 `translateY(40px)+opacity 0` → 제자리로, IntersectionObserver로 트리거. 살짝 stagger
5. **초기 로드 애니메이션**: 주요 요소들이 아래에서 위로 순차 등장(rise, delay 0.05s씩 증가)
6. **호버 인터랙션**: 인덱스 아이템 호버 시 좌측으로 살짝 이동 + 타이틀 시트론으로 + 하단에 시트론 언더라인이 좌→우로 채워짐
7. **`::selection`**: 드래그 선택 영역은 시트론 배경 + 잉크 텍스트
8. **스크롤 큐**: 모노 텍스트 + 가로 바에 wipe 애니메이션

**반응형**
- 820px 이하에서 그리드를 1열로 접고, 인덱스/통계 칸 수를 줄이고, 큰 들여쓰기 제거

**금지 사항**
- 둥근 카드 그림자, 그라데이션 버튼, 파스텔톤, 일반적인 SaaS 템플릿 느낌 ❌
- 시트론을 넓은 면적에 칠하지 말 것 — 액센트로만

## ⚡ 짧은 버전 (간단히 쓸 때)

> 다크 브루탈리스트 에디토리얼 스타일로 만들어줘. 따뜻한 검정 배경(#0b0b09), 크림색 텍스트(#f4f1e6), 형광 시트론 액센트(#ccff00) 딱 하나만. 본문은 Pretendard Medium, 라벨·메타·숫자는 JetBrains Mono 대문자. 헤딩은 weight 800에 clamp로 초대형, letter-spacing 타이트(-0.04em), line-height 0.86. 박스 대신 1px 헤어라인으로 구분. 섹션 라벨은 `001 — [라벨]` 번호 형식. 필름 그레인 오버레이, 시트론 커스텀 커서, 무한 마퀴 띠, 스크롤 리빌 애니메이션 포함. 둥근 카드·그림자·그라데이션·파스텔 금지.
